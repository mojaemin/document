10주요 트렌드

AW2026 전시회 데이터 기반 스마트팜 자동화 분야 핵심 트렌드
1. 협동로봇 고가반하중화
협동로봇 가반하중이 20~50kg급으로 확대. 두산로보틱스(25kg, 세계 1위), FANUC CRX-30iA(30kg), 뉴로메카 Indy(30/50kg급)가 주도. 안전 펜스 없이 대형 농산물 수확·운반 실현.
2. SW정의 PLC (소프트웨어 정의 자동화)
LS일렉트릭 SU-CM70은 기능을 SW 앱처럼 추가·변경. 모벤시스 WMX는 PC 기반 소프트 PLC로 59% 비용 절감. 온실 환경 제어 기능을 유연하게 확장 가능.
3. AMMR 복합로봇 (로봇팔+AMR 통합)
원익로보틱스 AMMR, 유진로봇 GoCart Omni+협동로봇, 마로로봇테크 모바일 매니퓰레이터. 온실 내 자율 이동+수확·운반·선별을 한 번에 수행하는 복합 작업 로봇 실용화 가속.
4. AI 비전 노코드화
뉴로클 Neuro-T(노코딩 오토딥러닝), 슈퍼브AI ZERO(제로샷, 라벨링 불필요), 코그넥스 In-Sight 3800(AI 엣지러닝). IT 인력 부족한 농업 현장에서 비전문가도 AI 모델 직접 구축.
5. 콜드체인 자동화
긱플러스 SkyCube(4배 밀도 냉장 자동창고), 아세테크 오토스토어(큐브형 냉장 물류), 현대글로비스 ORCA WCS(입고~출고 통합), 멀티웨이 무인지게차(냉장 저장고 자동 입출고).
6. 5G/IoT 기반 스마트팜 통합 플랫폼
KT 기가 스마트팜(5G+AI+IoT), 비바엔에스 잇츠스마트팜, 큐빅스 국가표준 센서(WiFi/LTE/LoRa), KETI IoT 표준 플랫폼(oneM2M). 실시간 병해충 감지, 원격 환경 제어, 자율 의사결정.
7. LLM/생성AI의 산업 자동화 침투
LS일렉트릭 LLM 대화형 진단, 싸이몬 AI Copilot(자연어 PLC 코딩), 로크웰 Copilot(AI 설계 자동화), 라온피플 LLM+VLM 비전. 비전문가도 자연어로 설비 진단·코드 생성.
