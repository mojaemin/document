## 1. 전시회 개요 및 조사 목적

### 전시회 개요

AW2026은 국내 최대 규모의 스마트공장·자동화산업전으로, 코엑스 전관(A·B·C·D홀)을 사용하여 24개국 500개 기업이 참가합니다.

**2026년 핵심 변화 — 주요 테마:**

| 테마 | 내용 |
| ------ | ------ |
| Physical AI / 자율제조(AX) | 로봇이 물리 환경 인식→자율 판단·동작, 올해 핵심 테마 |
| 휴머노이드 대거 등장 | Atlas(보스턴), TR-WORKS(티로보틱스), 중국 빅5 등 |
| 소프트웨어 정의 자동화(SDA) | LS SU-CM70, 슈나이더 EAE, 보쉬 ctrlX — SW로 기능 정의·확장 |
| LLM/자연어 AI 통합 | LS(LLM 진단), 싸이몬(AI Assistant), 로크웰(Copilot) |

**주요 전시 구역:**

| 전시관 | 주요 내용 |
| -------- | ---------- |
| A·B홀 | 로봇·자동화, Korea Vision Show, 협동로봇, 산업용 로봇 |
| C홀 | PLC·HMI·센서, AI 팩토리 특별관, 제어·계측 |
| D홀 | 스마트물류 특별관, AGV/AMR, 물류 자동화 |

### 조사 목적

**'로봇 기반 자동화 스마트팜 운영 플랫폼'** 구축을 위해 AW2026 참가업체 중 스마트팜에 적용 가능한 기업·기술을 **8개 기술 항목**으로 구분하여 조사:

```text
로봇 그리퍼 → 로봇팔/협동로봇 → AMR/AGF/AGV → AI 물류 자동화
→ AI 선별 자동화 → 비전 선별 기술 → AI PLC 통합 플랫폼 → 스마트팜 관련 기술
```

## 2. AI 병렬 에이전트 조사 방법론

AI 에이전트 8개를 병렬로 운영하여 각 기술 항목을 동시에 탐색했습니다:

| 에이전트 | 담당 분야 | 조사 범위 |
| ---------- | ---------- | ---------- |
| **Agent 1** | 로봇 그리퍼 | 소프트 그리퍼, 적응형 그리퍼, 진공 그리퍼, 로봇핸드 |
| **Agent 2** | 로봇팔/협동로봇 | 협동로봇, 6축/7축 로봇팔, SCARA, 델타로봇 |
| **Agent 3** | AMR/AGF/AGV | 자율이동로봇, 무인운반차, 무인지게차 |
| **Agent 4** | AI 물류 자동화 | 팔레타이징, 자동 창고, WMS, 물류 턴키 |
| **Agent 5** | AI 선별 자동화 | AI 딥러닝 비전 검사, 자동 분류, MLOps |
| **Agent 6** | 비전 선별 기술 | 머신비전, 3D 비전, 라인스캔, 하이퍼스펙트럴 |
| **Agent 7** | AI PLC 통합 플랫폼 | PLC, IoT, SCADA, MES, 디지털트윈 |
| **Agent 8** | 스마트팜 관련 기술 | 환경제어, 양액, 식물공장, 센서, 드론 |

**조사 소스:** AW2026 공식 사이트(automationworld.co.kr), 헬로티·로봇신문·전자신문·EBN 등 전시회 리뷰, 참가업체 프레스 보도, 산업 분석 자료
