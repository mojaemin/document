현재 프로젝트의 변경사항을 GitHub에 push하여 GitHub Pages에 배포해주세요.

1. `git status`로 변경사항 확인
2. 변경된 파일이 있으면 모두 스테이징 (`git add -A`)
3. 변경 내용을 요약하여 한국어 커밋 메시지 작성
4. `git commit` 후 `git push origin main`
5. 배포 완료 후, 아래 형식으로 배포 URL을 안내해주세요:
   - 기본 URL: https://mojaemin.github.io/document/
   - 프로젝트 루트에 있는 모든 폴더(`.github/`, `.claude/`, `.git/` 등 설정 폴더 제외)를 나열
   - 각 폴더를 `https://mojaemin.github.io/document/{폴더명}/` 형식으로 안내
   - 이번에 변경된 폴더는 `(업데이트됨)` 표시를 붙여서 구분
   - 변경되지 않은 폴더도 함께 표시하되, 변경된 폴더를 먼저 보여주세요
   - 루트 레벨 파일이 변경된 경우 기본 URL에 `(업데이트됨)` 표시합니다.
