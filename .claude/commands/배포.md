현재 프로젝트의 변경사항을 GitHub에 push하여 GitHub Pages에 배포해주세요.

1. `git status`로 변경사항 확인
2. 변경된 파일이 있으면 모두 스테이징 (`git add -A`)
3. 변경 내용을 요약하여 한국어 커밋 메시지 작성
4. `git commit` 후 `git push origin main`
5. 배포 완료 후, 변경된 파일이 속한 폴더를 감지하여 해당 폴더별 배포 URL을 안내해주세요:
   - 기본 URL: https://mojaemin.github.io/aw2026-report/
   - 변경된 파일의 최상위 폴더를 자동으로 감지하여 `https://mojaemin.github.io/aw2026-report/{폴더명}/` 형식으로 안내
   - 루트 레벨 파일이 변경된 경우 기본 URL을 안내
   - 여러 폴더가 동시에 변경된 경우, 변경된 모든 폴더의 URL을 각각 안내
   - `.github/`, `.claude/` 등 설정 폴더는 배포 URL 안내에서 제외
